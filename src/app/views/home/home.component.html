Angular2-Draggable

ngx-draggable-resize has Angular(ver >= 4.x) directives that make the DOM element draggable and resizable

Documents Github

| Directives | Demo | | ---------- | ---- | |`ngDraggable` | [demo](#/draggable/usage/basic) | |`ngResizable` | [demo](#/resizable/default) |

Installation

```bash # if you use npm npm install ngx-draggable-resize --save # or if you use yarn yarn add ngx-draggable-resize ```

Import

```diff + import {{'{'}} AngularDraggableModule } from 'ngx-draggable-resize'; @NgModule({{'{'}} imports: [ ..., + AngularDraggableModule ], }) export class AppModule {{'{'}} } ```
